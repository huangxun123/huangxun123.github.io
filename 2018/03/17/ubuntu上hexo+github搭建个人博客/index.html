ubuntu上hexo+github搭建个人博客

Posted on 2018-03-17

周末利用Hexo搭建了自己的个人博客，然后将博客托管在github上，这里我将简要介绍博客搭建的具体步骤，也算是一种经验积累，我的平台是ubuntu系统.

安装hexo

安装hexo相当简单，但在安装前，我们得先检查是否已经安装了下列应用程序

  • Node.js
  • Git

在终端分别运行

若出现版本信息，则程序已经安装，否则进行下列安装步骤

  1. 安装nodejs，使用官方推荐的方法：
  1. 安装git，这里我参考的教程是Git教程-廖雪峰的官方网站，教程简单易懂，因为我们的目的是搭建个人博客然后放到远程仓库上托管，所以这个教程我们只需要学习到远程仓库这节就行，如果你想继续学习git，教程的后半部分也是十分不错的

当上面两个应用程序成功安装后，即可用npm安装Hexo(npm是随同Nodejs一起安装的包管理工具，能够解决Nodejs代码部署上的很多问题)

但是npm是直接使用国外镜像，所以直接使用npm命令可能会导致下载速度很慢，我就碰到了这个问题，解决方法是使用国内的淘宝镜像，即执行上述命令前先执行下面的命令：
